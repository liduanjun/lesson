## Libc

我们学习的libc库是Linux 应用程序开发当中最基础，最庞大的库。
ubuntu 14.04 amd64 发行版 中用到的libc库 是由 Gnu C libraries 实现的  
在bash中运行下面命令就可以得到 libc的信息

> /lib/x86_64-linux-gnu/libc.so.6
