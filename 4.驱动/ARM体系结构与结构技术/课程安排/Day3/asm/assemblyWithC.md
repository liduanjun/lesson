# GNU汇编语音与C语言混合编程

* C文件和汇编文件混合编译
* C语言内嵌汇编

## C文件和汇编文件混合编译

.c 文件 和 .s 文件分别编写，编译的时候都用 gcc 编译，编译成 .o 文件，
然后用 ld 工具将 .o 文件们 链接成一个程序文件

## C语言内嵌汇编

.c 文件里这么写汇编

```
asm volatile("汇编指令1\n\t"
        "汇编指令2\n\t"
        "汇编指令3\n\t"
        "汇编指令n"
        : 输出变量列表
        : 输入变量列表
        : 被破坏的寄存器列表);
```
